Etiquetas form,input,select,option

Asi como en el mundo fisico existen los libros, revistas y todos los tipos de impresiones pasivas,
existen tambien los formatos pre impresos en los que se le pide a una persona completar cierta informacion para luego
ser procesada son los llmadaso formularios


La etiqueta form 

sirve para agrupar el conjunto de items que van a ser procesados simultaneamente al ser transmitidos. 
visualmente, esta etiqueta no tiene ninguno efecto en la pantalla


Los atributos principales de la etiqueta form son los siguientes:

name:  nombre de formulario, el cual es muy importante si deseas manipular el 
comportamiento de la pantalla usando JavaScript
action:  Nombre de la pagina a la que se le habran de enviar los datos contenidos 
en el formulario para ser procesados
method: atributo opcional que sirve para indicar si el metodo usado en la llamdaa 
habra de ser GET o POST

las etiquetas select y option

este par de etiquetas sirven para mostrar en pantalla un 
cuadro combinado,mejor conocido como combo box



Los principales atributos de la etiqueta select son los siguientes

name:  nombre del elemento. Este atributo es fundamental ya que al 
ser enviado el formulario, este es el nombre mediante el cual se porda recuperar 
la opcion seleccionada por el usuario.

disabled:  si este atributo aparece dentro de la etiqueta del select, 
entonces el elemento aparecera como deshabilitado. Este atributo no necesita 
ser igualadao a ningun valor
Multiple:  si esta opcion aparece en la etiqueta, el comportamiento del 
elemento cambia significativamente, ya que el usuario tendra la opcion de seleccionar
mas de una opcion a la vez
Size:  permite determinar el numero de elementos que podran ser mostrados 
simultaneamente cuando el usuario este haciendo una seleccion



Los atributos principales de las etiquetas option son:

value:  indica el valor que habra de tomar el select en caso de que esta opcion sea seleccionada
selected:  indica que la opcion debe estar seleccionada inicialmente al mostrarse la pagina
