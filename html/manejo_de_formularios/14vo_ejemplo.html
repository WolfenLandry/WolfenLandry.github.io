La etiqueta textarea

    
Descripcion:

    esta etiqueta es un conjunto con su respectiva etiqueta de cierre, muestra en pantalla un 
    area de texto multilinea para que el usuario escriba un contenido potencialmente largo. 
    Todo lo que el programador escriba entro los tags de inicio y fin del textarea
    apareceran como contenido inicial

    
Los atributos principales de un  son los siguientes


        name: nombre del elemento.
        disabled: indica si el elemento aprecera como deshabilitado. Este atributo no necesita 
        ser igualado a ningun valor.
        Cols: numero de columnas visibles (caracteres).
        Rows: numero de filas visibles.
        Readonly: si este atributo aparece escrito indica que el usuario no podra alterar el contenido del elemento.
