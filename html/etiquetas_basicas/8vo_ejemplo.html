La etiqueta <'font>
    Las Etiquetas <'font> y  sirven para cambiar localmente el tipo y el color de la fuente
    utilizada para mostrar el texto escrito en medio de ambas
 
    Los principlaes atributos son:
    face: indica el nombre de la fuente por ejemplo Arial o Courier o Tahoma
    size: indica el tamaño de los caracteres
    color: indica el color que tendra el texto
