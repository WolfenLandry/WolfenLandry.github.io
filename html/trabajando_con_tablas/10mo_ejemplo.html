Existen un par de atributos que pueden darle mas poder a nuestras tablas, se usan para poder 
unir dos o mas filas o dos o mas columnas. se llaman rowspan y colspan

El atributo rowspan se usa para unir filas en una sola  
El atributo colspan se usa para unir columnas en una sola
